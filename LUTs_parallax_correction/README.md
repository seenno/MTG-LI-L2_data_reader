# Parallax and light travel time correction lok-up tables.

Download the light-travel-time and parallax correction look-up tables from the EUM server located at
https://sftp.eumetsat.int/public/folder/UsCVknVOOkSyCdgpMimJNQ/User-Materials/MTGUP/LI_parallax_correction/. 
Only the file named `SSP_0_LUT.tar.gz` needs to be downloaded and extracted in `./LUTs_parallax_correction/`
subdirectory (will result in 12 monthly files with a total size of ~2.4GB).

For running the tool successfully, the following files need to be in this directory:

`PX_Apr_2005_2018_MK_res1.nc`

`PX_Aug_2005_2018_MK_res1.nc`

`PX_Dec_2005_2018_MK_res1.nc`

`PX_Feb_2005_2018_MK_res1.nc`

`PX_Jan_2005_2018_MK_res1.nc`

`PX_Jul_2005_2018_MK_res1.nc`

`PX_Jun_2005_2018_MK_res1.nc`

`PX_Mar_2005_2018_MK_res1.nc`

`PX_May_2005_2018_MK_res1.nc`

`PX_Nov_2005_2018_MK_res1.nc`

`PX_Oct_2005_2018_MK_res1.nc`

`PX_Sep_2005_2018_MK_res1.nc`
